# Stratégie fundraising

## Phase 1 (Pre-seed)
- Objectif: MVP sécurisé + premières transactions réelles
- Cibles: angels fintech locaux + diaspora + micro-funds africains

## Phase 2 (Seed)
- Objectif: scaling B2C/B2B et conformité renforcée
- KPIs: CAC, volume transactionnel, taux rétention, défaut fraude

## Story d'investissement
- Inclusion financière
- Stack modulaire scalable
- Risque contrôlé (sécurité/compliance by design)
